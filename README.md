# BTG Communication
This is the new website for BTG Communication running with NextJs APP router & Bun as javascript runtime.

## Getting Started

First, run the development server:

```bash
bun run dev
# or
yarn dev
# or
pnpm dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
